# Open Science Funding DAO

A decentralized platform for funding scientific research through community token staking and milestone-based audits.

## 🚀 Features

- Create research proposals with multiple milestones
- Stake tokens to fund research
- Milestone completion and fund distribution
- Community-driven research funding
- Secure stake withdrawal system

## 🛠️ Technical Details

- Smart contract written in Clarity
- Comprehensive test suite
- Security features:
  - Access control checks
  - Input validation
  - Safe token transfers
  - Admin controls

## 📋 Usage

1. Deploy contract:
```bash
clarinet contract deploy
```

2. Create proposal:
```bash
clarinet contract call create-proposal
```

3. Stake tokens:
```bash
clarinet contract call stake-tokens
```

## 🧪 Testing

Run the test suite:
```bash
clarinet test
```
